# Tables

This folder contains all tables generated by the R pipeline (CSV format).

Run `source("R/ethnomathematics_bibliometric_analysis.R")` to generate them.

| File | Description |
|---|---|
| `table_01_summary.csv` | Corpus summary statistics |
| `table_02_doc_types.csv` | Document type distribution |
| `table_03_top_sources.csv` | Top 10 publication sources |
| `table_04_top_countries.csv` | Top 15 countries |
| `table_05_top_authors.csv` | Top 15 authors |
| `table_06_top_keywords.csv` | Top 20 author keywords |
| `table_07_gam_statistics.csv` | GAM model statistics |
| `table_08_macro_clusters.csv` | Thematic macro-clusters summary |
